# vue-setup
the repo is a demo for vue 2.7, which has backported some features (eg. setup) from vue 3.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
